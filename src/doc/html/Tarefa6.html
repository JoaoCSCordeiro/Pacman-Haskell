Description

Introdução

O objetivo desta tarefa é automatizar por completo o jogador Pacman.

Solução

Para isto foi usada uma estratégia similar à usada na tarefa 5 em que é usada a função ifpacMega que deteta o estado do Pacman, se este for Mega ele vai procurar o movimento ótimo para perseguir os fantasmas usando a função aproximaPac, caso contrário é usada a função fogepac para que o Pacman fuga dos fantasmas de modo e evitar a sua morte.

Conclusão

Mesmo achando que tinhamos a ideia certa não foi possivel implementar um robô funcional que jogassse Pacman automaticamente.
